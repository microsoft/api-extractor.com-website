<p>&nbsp;</p>

```
usage: api-extractor init [-h]

Use this command when setting up API Extractor for a new project. It writes
an api-extractor.json config file template with code comments that describe
all the settings. The file will be written in the current directory.

Optional arguments:
  -h, --help  Show this help message and exit.

```
